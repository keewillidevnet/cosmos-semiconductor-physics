# Figures

## Architecture Diagrams

This directory will contain architecture diagrams and visualizations for CSP.

### Planned Figures

1. **csp-architecture-overview.png** - High-level system architecture
2. **physics-encoder-detail.png** - Detailed physics encoder architecture
3. **cross-domain-attention.png** - Cross-domain attention mechanism
4. **process-flow-transformer.png** - Process flow transformer architecture
5. **training-data-pipeline.png** - Data pipeline architecture
6. **validation-workflow.png** - Multi-stage validation process

### Figure Specifications

- Format: PNG (diagrams), SVG (vector graphics)
- Resolution: 300 DPI minimum
- Style: Consistent with NVIDIA technical documentation
- Color scheme: NVIDIA green (#76B900) accent

### Creating Figures

Figures can be created using:
- draw.io / diagrams.net (recommended for architecture)
- Matplotlib/Seaborn (for data visualizations)
- TikZ/LaTeX (for publication-quality figures)
